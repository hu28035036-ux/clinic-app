🐞 문자나라 한글 깨짐 수정 (중요 버그)
  - SMS 발송 요청: GET → POST 변경
      · 이전 GET 방식은 한글 'message' 를 URL 쿼리스트링에 넣어 일부
        통신사·단말에서 깨진 문자로 전달되는 경우 발생
      · POST form-urlencoded + Content-Type charset=utf-8 헤더로 전환
  - SMS 응답 디코딩: 서버 Content-Type charset 우선, UTF-8/EUC-KR/CP949
    순 자동 폴백 (_smart_decode_response 헬퍼 신설)
      · 이전 "decode utf-8 errors=ignore" 는 서버가 EUC-KR 로 응답할 때
        한글 바이트가 전부 무시돼 오류 메시지가 빈 문자열이 되던 문제 수정

🔄 자동 업데이트 기능 완성 (버튼 3클릭으로 최신화)
  - /api/about/download-update : ZIP 다운로드 + SHA256 검증
  - /api/about/apply-update    : updater.bat 실행 + 자기 종료
  - updater.bat                : 본체 종료 대기 → 파일 교체 → 재시작
                                 실패 시 .old 파일로 자동 롤백
  - 관리자 › 시스템 › 버전 카드 에 다운로드/설치 버튼 + 진행률 표시

🛡️ 업데이트 전 DB 자동 백업 (안전망)
  - "지금 설치" 버튼 클릭 직전에 SQLite backup API 로 DB 스냅샷 생성
  - 파일명: clinic_before_update_v<버전>_<타임스탬프>.db
  - 저장 위치: %APPDATA%\도수치료예약\backups\
  - 문제 생기면 복원하기.bat 또는 파일로 복원 UI 로 즉시 복구
  - 백업 실패해도 업데이트는 진행 (안전망일 뿐, 차단 X)

✨ SHA256 입력 방어 처리

  - 공백/탭/개행 정리 — 복사 붙여넣기 실수 방지
