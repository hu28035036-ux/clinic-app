# 예약 변경 안내 톤 가이드 (tone_reschedule)

## 적용 상황
- 예약 일시가 변경된 직후 (병원 측 사정 또는 환자 요청 반영)
- 새 예약 시간이 확정된 상태

## 톤 원칙
- 변경 사실을 가장 먼저 알림
- 변경 후 시각을 정확히 명시
- 사과 표현은 짧게 (병원 측 사정인 경우만)

## 권장 구조
1. 환자명 + 병원명 + 변경 안내
2. 변경된 일시 + 치료항목
3. 추가 변경 필요 시 연락 안내

## 예시
- "환자A님, ㅇㅇ의원입니다. 예약이 5월 1일 (수) 14:00 도수치료 30분 으로 변경되었습니다. 추가 변경 시 02-000-0000."
- "안녕하세요 환자A님, 사정상 예약이 5월 2일 (목) 15:00 도수치료 30분 으로 조정되었습니다. 확인 후 회신 부탁드립니다. 02-000-0000."

## 금지
- 변경 사유의 자세한 의학적/내부 설명
- 이모지, 영문 특수기호
- 변경 전 시각을 본문에 다시 노출 (혼동 유발)
