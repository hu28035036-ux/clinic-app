# 치료사 휴무 등록 (therapist_leave)

## 적용 상황
- 특정 치료사의 연차·반차·휴무를 등록해서 그 시간에 예약이 잡히지 않게 할 때
- 등록된 휴무를 수정·삭제할 때

## 위치
- 메인 화면 상단 탭에서 "직원" 탭 선택
- 직원 탭 안의 "휴무일 관리" 하위 탭(sub-tab)을 연다

## 절차 — 휴무 등록
1. "직원" 탭 → "휴무일 관리" 하위 탭을 연다.
2. 캘린더에서 휴무를 등록할 날짜를 선택한다.
3. 대상 치료사를 선택한다.
4. 휴무 유형을 고른다:
   - 종일 휴무
   - 오전 반차 (오전 시간대만 휴무)
   - 오후 반차 (오후 시간대만 휴무)
5. (선택) 사유나 메모를 입력한다.
6. 저장 버튼을 누른다.

## 절차 — 휴무 수정·삭제
1. "휴무일 관리" 화면 캘린더에서 등록된 휴무가 표시된 날짜를 클릭한다.
2. 해당 날짜의 휴무 목록에서 수정·삭제할 항목을 선택한다.
3. 수정: 유형/메모를 바꾼 뒤 저장. 삭제: 삭제 버튼을 누른다.

## 시스템 동작
- 휴무가 등록되면 그 시간대에 해당 치료사로 예약을 새로 잡을 수 없다.
- 이미 잡혀 있던 예약이 휴무와 겹치면 시스템이 경고로 표시한다 — 별도로 예약을 옮기거나 다른 치료사로 재배정한다.
- 오전/오후 반차의 경계 시각은 시스템 설정값(예: 오전 14:00 까지 / 오후 13:00 부터)을 따른다.

## 주의
- 다른 직원의 휴무를 임의로 수정하지 않는다. 본인 또는 권한이 있는 사람만 등록·변경한다.
- 등록 후 캘린더 화면이 즉시 갱신되지 않으면 다른 날짜를 한 번 클릭한 뒤 다시 돌아오면 반영된다.
