# AI 기능 설정 (ai_settings)

## 적용 상황
- AI 도우미, 예약문자 AI 초안 같은 AI 기능을 처음 켤 때
- API Key 를 새로 발급받았거나 모델을 바꿀 때
- AI 기능이 동작하지 않아 점검할 때

## 위치
- 메인 화면 상단 탭에서 "관리자" 탭 선택
- 관리자 탭 안의 "AI 설정" 하위 탭을 연다

## 절차 — AI 기능 켜기
1. "관리자" 탭 → "AI 설정" 하위 탭을 연다.
2. "AI 기능 ON/OFF" 토글을 ON 으로 바꾼다. 기본값은 OFF 이다.
3. Provider 를 선택한다 (OpenAI 또는 Anthropic).
4. 모델명을 입력한다 (예: gpt-4o-mini, claude-haiku-4-5 등 — 선택한 provider 에서 사용 가능한 모델명).
5. API Key 를 입력한다. 저장 후에는 앞 4자리만 화면에 보이고 나머지는 마스킹된다.
6. (선택) max_tokens, temperature 등을 조정한다. 기본값으로 시작해도 무방하다.
7. "저장"을 누른다.

## 절차 — 상태 점검
1. AI 설정 화면 하단 또는 상태 박스에서 다음을 확인한다:
   - AI 기능: ON
   - API Key: 설정됨
   - 모델: 입력됨
   - SDK: 설치됨
   - Ready: 예
2. "Ready: 예" 가 아니면 부족 항목을 다시 확인한다.

## API Key 변경·삭제
- 새 키 입력 후 저장: 기존 키가 새 키로 교체된다.
- 키를 비워 두고 저장: 기존 키가 그대로 유지된다 (실수로 지우는 것을 방지).
- 키를 명시적으로 지우려면 "API Key 삭제" 옵션을 사용한다.

## 주의
- API Key 는 절대 외부로 공유하지 않는다. 화면에서는 마스킹되어 보이지만 실제 값은 평문으로 저장된다.
- AI 가 직접 환자 데이터를 수정하거나 문자를 자동 발송하지 않는다 — 모든 결과는 사람의 검토를 거친다.
- AI 가 꺼져 있어도 예약·문자·통계 같은 기본 기능은 정상 동작한다. AI 기능만 비활성화될 뿐이다.
- AI 도우미(업무 매뉴얼 Q&A)도 같은 AI 설정을 사용한다. AI 기능이 꺼져 있으면 도우미도 답변하지 않는다.
