[주요 변경 (v1.3.1) — 보안 + 안정성 패치]

  🛡️ 자동 업데이트 hang 수정
     · updater.bat: rename 5회 재시도 + ExecutionPolicy Bypass + rollback pause
     · apply_update: DETACHED_PROCESS, /k→/c, sleep 3초, 백업 실패 시 중단
     · 신규 GET /api/about/update-log — 진단용 로그 조회
     · 프론트: 30/60초/6분 단계별 안내 + 로그 모달 + 자동 새로고침

  🛡️ 인증 누수 회귀 수정 (HIGH)
     · /api/sync/pull, /sync/push, /sync/now, /backup 에 require_admin 추가
     · /api/mode 첫 실행 시 인증 우회 차단
     · sync pull/push 듀얼 인증 (관리자 토큰 OR X-Sync-Token)
       - config.sync_secret 자동 생성 + sync_with_peer 가 헤더 동봉
       - GET /api/config 에서 secret 자동 제거 (이전 회귀: 익명 노출 가능)
       - 관리자 전용: GET /api/config/sync-secret, POST /regenerate-sync-secret

  🛡️ DB 복원 안전성
     · 임시 파일 → integrity_check → atomic 교체
     · engine.dispose() + db Session 의존성 제거 — Windows 파일 잠금 회피
     · 손상된 파일 업로드 시 운영 DB 무손상 보장

  🛡️ SMS 발송 비밀 노출 차단
     · 서버 echo / urllib 예외 메시지에 platext passwd/key 자동 마스킹

  ⚙️ UX
     · 데이터 변환 spinner + 버튼 비활성화 + 90초 타임아웃
     · 모달 액션 버튼 sticky (긴 모달 스크롤 시 항상 가시)

  ⚙️ 빌드 인프라
